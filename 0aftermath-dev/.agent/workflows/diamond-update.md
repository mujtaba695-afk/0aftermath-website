# Diamond Standard Update Workflow

1. **Reconnaissance**: Use the SEO Oracle persona to identify content gaps or technical issues.
2. **Design**: Use the UI Architect persona to propose visual changes or new assets.
3. **Implementation**: Use the Lead Developer persona to write code in the staging environment.
4. **Local Verification**: Run a local server and verify:
   - Performance (Lighthouse).
   - Responsive design.
   - Interactive elements.
5. **Critique**: Self-audit based on "The Critic" (Validation) guidelines.
6. **Walkthrough**: Generate a detailed walkthrough artifact for the user.
7. **Sync**: Upon approval, sync staging code to the production folder.
